EOG ASSIST: EYE MOVEMENT COMMUNICATION SYSTEM

Version 1.0 | Final Project Presentation

1. Project Summary

Clinical Problem: Patients with reduced mobility or severe speech limitations cannot communicate basic needs efficiently.

Value Proposition: A non-invasive, low-cost system using electrooculography (EOG) to translate eye movements into real-time IoT commands.

Last Update: April 13, 2026

2. Device & Sensor Description

Hardware Components

  • Instrumentation Amplifier: AD620
  • Operational Amplifiers: TL084 / LM741
  • Microcontroller: Arduino

Technical Specs

  • Variable: Ocular Electrical Potential (EOG)
  • Range: 10 µV to 5 mV
  • Sampling: 250 Hz (Nyquist compliant)
  • Power: < 500 mW (Low consumption)

Connectivity Strategy

The system includes a local buffer for connectivity loss, ensuring delayed synchronization with the Grafana cloud platform once Wi-Fi is restored.

4. Technical IoT Architecture

Perception Layer: Ag/AgCl Surface Electrodes and AD620 differential gain stage.

Connectivity Layer: Wi-Fi via MQTT/HTTP protocols.

Processing Layer: Edge computing for thresholding; Cloud computing for visualization.

Application Layer: Grafana Dashboard with real-time caregiver alerts.

5. Rule-Based Logic vs. AI

The project specifically avoids Artificial Intelligence to prioritize:

  • Transparency: Avoiding "Black Box" algorithms for clinical safety.
  • Efficiency: Linear gaze-voltage relationship (< 30°) allows simple rule-based triggers.
  • Cost: Minimal computational load for real-time edge processing.

Clinical Evidence: Supported by AASM standards regarding horizontal gaze polarities.

6. Users & Ethics

User Profile

Primary: Bedridden patients with motor disabilities.

Secondary: Caregivers and family members at home.

Ethical Compliance

  • IEC 60601 (Electrical Safety)
  • ISO 13485 (Quality Management)
  • Informed Consent processes

7. Scope & Roadmap

In Scope

Detection of horizontal EOG movements, Wi-Fi communication, and real-time visualization in Grafana.

Out of Scope

Official medical certifications (FDA/COFEPRIS), commercial PCB manufacturing, and native mobile apps (due to time and resource constraints).

Future Roadmap

Miniaturization into a wearable device and integration of computer vision for electrode-free detection.
